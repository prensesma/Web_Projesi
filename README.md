# Web_Projesi
## proje açıklaması
Bu proje, kullanıcıların film ve dizileri gelişmiş filtreleme sistemi ile keşfedebildiği, IMDb puanlarını görüntüleyebildiği, fragman izleyebildiği ve kişisel izleme listeleri oluşturabildiği etkileşimli bir web uygulamasıdır. Kullanıcılar farklı kriterlere göre arama yaparak kendilerine en uygun içeriği hızlı ve pratik şekilde bulabilir.
## hedef kullanıcılar 
film ve dizi izleyicileri
dijital platform kullanıcıları
içerik önerisi arayan kişiler
## temel özellikler
kullanıcı kayıt ve girişi
film ve dizi listeleme
gelişmiş filtreleme (tür, yıl, IMDb puanı, platform)
fragman izleme
favorilere ekleme
izledim / izleyeceğim listesi
yorum ve puanlama sistemi
## kullanılan teknolojiler
html
css
javaScript
php
mysql
